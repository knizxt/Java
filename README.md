# Advanced Java Algorithms Repository

This repository contains a collection of 150+ high-complexity Java source files implemented for educational and competitive programming purposes. The collection covers a wide range of advanced algorithms, data structures, and mathematical concepts.

## Project Overview
The goal of this project is to provide a robust library of algorithmic templates in Java. Each file is structured to include a class name corresponding to the problem and a basic execution structure.

## Categories

### 1. Advanced Graph Algorithms
Includes implementation of:
- **Shortest Paths**: Dijkstra, Bellman-Ford, Floyd-Warshall.
- **Flow & Matching**: Dinic's Algorithm, Edmonds-Karp, Hopcroft-Karp.
- **Decomposition**: Heavy-Light Decomposition, Centroid Decomposition.
- **Connectivity**: Tarjan's SCC, Articulation Points, Bridges.

### 2. Dynamic Programming
Advanced DP techniques covering:
- **Optimization**: Bitmask DP (TSP), Digit DP.
- **Tree DP**: Diameter and path optimizations.
- **Interval DP**: Matrix Chain Multiplication, Burst Balloons.
- **String DP**: Longest Common Subsequence, Edit Distance.

### 3. Advanced Data Structures
Implementations of:
- **Trees**: Segment Trees (Lazy Propagation), Fenwick Trees, Tries, AVL/Red-Black Trees.
- **Advanced Heaps**: Fibonacci Heap, Binomial Heap.
- **Spatial**: QuadTree, KD-Tree.
- **Probabilistic**: Bloom Filter.

### 4. String Algorithms & Geometry
- **Matching**: KMP, Rabin-Karp, Z-Algorithm, Aho-Corasick.
- **Structures**: Suffix Arrays, Suffix Automaton.
- **Geometry**: Convex Hull (Graham Scan), Point in Polygon, Line Intersection.

### 5. Backtracking & Advanced Mathematics
- **Search**: N-Queens, Sudoku Solver, Knight's Tour.
- **Math**: FFT (Fast Fourier Transform), Pollard's Rho (Factorization), Miller-Rabin (Primality Test).
- **Number Theory**: Lucas Theorem, Chinese Remainder Theorem, Matrix Exponentiation.

## Complexity Highlights
Each problem is chosen to represent a "Medium-Hard" or "Hard" level in competitive programming (equivalent to LeetCode Hard or Codeforces Div 2 D/E/F).

## Usage
Each file is a standalone Java class. To run a specific algorithm:
1. Navigate to the `Java` directory.
2. Compile the desired file: `javac <FileName>.java`.
3. Run the class: `java <FileName>`.

Example:
```bash
javac Dijkstra.java
java Dijkstra
```
